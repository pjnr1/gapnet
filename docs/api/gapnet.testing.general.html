Matches the number in the format $.$ms

If multiple $.$ms exists, the last one will be used.

Example:

    /somefolder/gdt__12.5ms.pt              -> 12.5
    /somefolder/51.3/542.2ms/gdt__2.5ms.pt  -> 2.5
